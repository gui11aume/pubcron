PubCron Help

To use PubCron, you need to have a Google account with a valid address. PubCron is powered by Google App Engine, which takes care of data storage and Internet security issues. In addition, the PubCron mail display is optimized for Gmail. If you want the PubCron mail to be sent to another address, you will have to set up a mail redirection.

Then, you have to fill in a valid PubMed query. The specified term will be queried every (week) day and the hits will be sent to your address. PubMed queries have their specific syntax which is explained here.

PubCron allows you to give relevance feedback, so that it learns your areas of interest and ranks them higher in the future. The articles with the highest relevance score will appear first in the PubCron mail.

PubCron is free and open source. It is written in Python and the code is available at Github.
